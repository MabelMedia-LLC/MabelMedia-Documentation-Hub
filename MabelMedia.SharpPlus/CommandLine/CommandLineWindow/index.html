    CommandLine.CommandLineWindow

    Fields

    • string Title - The Window's Title Text.
    • bool Visible - Determines Wether Or Not To Render The Window.
    • Rectangle Bounds - The Window's Size And Position.
    • string[] Content - The Window's Content.
    • List<HotKey> HotKeys - The Window's Bound HotKeys.

    Methods

    • new(string Title, int X, int Y, int Width, int Height, string[] Content, List<HotKey> HotKeys, [bool Visible = true]) - Create A New CommandLineWindow.
    • void Render() - Render The Window.
    • void HotKeyHandler(ConsoleKeyInfo Key) - Handle The Current Key Press, Using The Window's Bound HotKeys.
    • void Close() - Close The Window.
